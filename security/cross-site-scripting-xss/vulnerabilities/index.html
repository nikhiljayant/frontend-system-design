What is XSS?

XSS stands for Cross-Site Scripting. It is a security vulnerability where a website allows users to input JavaScript code that is later rendered to other users. Since browsers run any JavaScript they see inside HTML, they execute the hacker's code. This allows hackers to steal information (like session cookies) or modify the website's look.
