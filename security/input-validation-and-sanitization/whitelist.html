Whitelist Validation

Whitelist validation accepts only explicitly permitted characters and rejects everything else. This is highly effective against command and script injections.

Enter username to test validation...

How it works:

We use a simple regular expression /^[a-zA-Z0-9_]{3,16}$/ to whitelist permitted characters. If the input matches, it is valid; otherwise it is rejected. 

// Strict whitelist check
const usernameRegex = /^[a-zA-Z0-9_]{3,16}$/;
const isValid = usernameRegex.test(inputValue);
