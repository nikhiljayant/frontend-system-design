Media Validation

Validating files ensures uploaded items match expected MIME types and remain within safe size constraints to prevent resource exhaustion.

How it works:

We read the file.type and file.size properties of the selected file, verify that the type is in our whitelist, and that the size does not exceed our maximum bound (1 Megabyte).
