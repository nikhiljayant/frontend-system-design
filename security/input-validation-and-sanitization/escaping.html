HTML Output Escaping

Escaping converts raw characters into HTML entity representations. This makes scripts inert, preventing XSS execution in browsers.

How it works:

We replace unsafe characters < > & " ' / with their corresponding HTML entity codes before placing them in the DOM.

// Safe HTML encoding
function escapeHTML(str) {
    return str.replace(/[<>"'/&]/g, (m) => {
        const map = {
            '&': '&amp;', '<': '&lt;', '>': '&gt;',
            '"': '&quot;', "'": '&#x27;', '/': '&#x2F;'
        };
        return map[m] || m;
    });
}
